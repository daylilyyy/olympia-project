# Olympia Project

## 프로젝트 소개

본 프로젝트는 해외 유명 보디빌더의 프로필을 보기쉽게 안내하기 위한 홈페이지입니다.
**React와 Tailwindcss를 연습하기 위한 toy project**입니다.
